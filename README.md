# VERY EARLY ALPHA!

## Build

### Linux
- download 
	- libsdl2-2.0-0
	- libsdl2-image-2.0-0
	- libsdl2-2.0-dev
	- libsdl2-image-2.0-dev
	- lua5.3
	- liblua-5.3-0
	- liblua-5.3-dev
	
- git clone this project
- cd into project and mkdir /build directory
- cd into /build and compile as cmake ..
- run make
